# Sharing App

## Description 
A file-sharing application that allows you to upload any type of file (.pdf, .png, .jpg, .rar, etc.) and share the download URL (which expires after 24 hours) with your friends

### Stack used
- Nodejs
- express
- MongoDB
- sass
- HTML
- js
